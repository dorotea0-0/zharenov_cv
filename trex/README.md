## Что это

DinoBot — это программа, которая автоматически играет в игру с динозавром в Google Chrome (та самая, что появляется когда нет интернета). 

Использует компьютерное зрение (OpenCV) для анализа экрана и принимает решения в реальном времени: когда прыгать, когда приседать, когда ускорять приземление.

**Зачем это нужно:**
- Интересно с технической точки зрения
- Можно похвастаться рекордом
- Пусть бот фармит очки, пока ты занимаешься делами
- Просто потому что можно

## Возможности

✅ **Автопоиск игры** — находит игру на экране автоматически  
✅ **Распознавание препятствий** — видит кактусы (одиночные, группы, большие) и птиц  
✅ **Адаптивная скорость** — подстраивается под ускорение игры  
✅ **Fast Fall** — техника быстрого приземления для точных прыжков  
✅ **Авто-рестарт** — если врезался, сам начинает заново  
✅ **Защита от ложных срабатываний** — не прыгает на каждый пиксель  
✅ **Отладочный режим** — видит, что «думает» бот в реальном времени  
✅ **Мониторинг FPS** — контроль производительности
